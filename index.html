Stock Market Project

This is a Java project simulating a stock market, featuring classes like Stock Market Project and Flower Shares. It builds a simple stock trading environment where users can interact with shares and manage purchases through a GUI. The code handles stock transactions, displays information, and simulates market activities. The project demonstrates practical use of Java object-oriented programming concepts and graphical user interface design, making it a great showcase for applying Java skills in a real-world scenario.

Intro Web Scraper

The "Public Web Scraper" is a Python project designed to demonstrate the basics of web scraping by targeting manhwa (as an example) sites. It features critical scripts like manhwa graber for data extraction and web scap chaplist for managing chapter lists. This project showcases practical scraping techniques, providing a real-world coding example perfect for those learning Python and web scraping.

AI Driven Test Generation

HPCG Testing and Artifact Generation is an ongoing research effort into using new and different tools in code development. This includes but is not limited to seeing how different AI tools perform better for specific tasks and how you can change statements to provide the best results for the project you are working on.
