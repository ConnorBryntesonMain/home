Stock Market Project

This is a Java project simulating a stock market, featuring classes like StockMarketProject.java and FlowerShares.java. The code handles stock transactions, displays information, and simulates market activities. The project demonstrates practical use of Java object-oriented programming concepts and graphical user interface design, making it a great showcase for applying Java skills in a real-world scenario.
